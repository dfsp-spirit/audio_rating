How it works:

  • Enter any memorable text to generate a unique 20-character ID
  • This ID will always be the same for your input text
  • The next time you participate, just come back and enter the same text again to get your link including the ID
  • Optionally, you can save the ID or link yourself. Do not pass the link to others.
  • If you want to invite friends to join, just send them a link to this page.

Privacy Information:

  • Local generation: The ID is generated in your browser only
  • One-way process: Your input is converted using a mathematical function that cannot be reversed - it's impossible to recover your original text from the ID
  • No data storage: Your input text is never sent to or stored anywhere
  • Purpose: The unique ID allows us to link multiple submissions from the same person without storing personal data like email addresses, usernames and passwords for open studies
