How it works:

  • Enter any unique memorable text to generate a unique ID from it
  • This ID will always be the same for the same input text
  • If you forget your ID, just come back and enter the same unique text again
  • Note: This is not a password or authentication method, just a way to optionally link your submissions together if you want to

Privacy Information:

  • Local generation: The ID is generated in your browser only
  • One-way process: Your input is converted using a mathematical function that cannot be reversed - it's impossible to recover your original text from the ID
  • No data storage: Your input text is never sent to or stored anywhere
  • Only ID stored: If you submit the audio ratings from your unique link, we will save the ID, but never you input text
  • Purpose: The unique ID allows us to link multiple submissions from the same person without storing personal data like email addresses
